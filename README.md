# Company Valuation Platform

Платформа для итеративной оценки стоимости компании на основе открытых источников данных.

## Возможности

- **Итеративный сбор данных** - с каждой итерацией обнаруживаются новые источники
- **Множество источников** - сайт компании, WHOIS, социальные сети, новости, вакансии, технологии
- **Автоматическая оценка** - расчет стоимости на основе нескольких методов
- **Отчеты в DOCX** - профессиональные отчеты в формате Word
- **Интерактивный дашборд** - HTML-дашборд с графиками и метриками

## Установка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd ai-research

# Установить зависимости
pip install -r requirements.txt
```

## Использование

### Базовый запуск

```bash
python -m company_valuation --domain example.com
```

### С указанием количества итераций

```bash
python -m company_valuation --domain stripe.com --iterations 5
```

### Полный набор опций

```bash
python -m company_valuation \
    --domain company.io \
    --iterations 3 \
    --output ./reports \
    --include-raw-data \
    --verbose
```

### Параметры командной строки

| Параметр | Сокращение | Описание | По умолчанию |
|----------|------------|----------|--------------|
| `--domain` | `-d` | Домен компании (обязательный) | - |
| `--iterations` | `-i` | Количество итераций сбора данных | 3 |
| `--output` | `-o` | Директория для отчетов | ./output |
| `--no-docx` | - | Не создавать DOCX отчет | false |
| `--no-dashboard` | - | Не создавать HTML дашборд | false |
| `--include-raw-data` | - | Включить сырые данные в DOCX | false |
| `--verbose` | `-v` | Подробный вывод | false |
| `--quiet` | `-q` | Минимальный вывод | false |

## Источники данных

### Итерация 1 (базовая)
- Сайт компании (заголовок, описание, контакты)
- WHOIS данные (возраст домена, регистратор)
- Технологический стек (фреймворки, инфраструктура)

### Итерация 2+
- Социальные сети (LinkedIn, Twitter, GitHub и др.)
- Новости и пресс-релизы
- Вакансии (рост компании)
- Финансовые данные (для публичных компаний)

## Методы оценки

1. **Рыночная капитализация** - для публичных компаний
2. **Мультипликатор выручки** - отраслевые коэффициенты
3. **По количеству сотрудников** - типичная стоимость на сотрудника
4. **По раундам финансирования** - оценка на основе инвестиций

## Ограничения

- Данные собираются из публичных источников и могут быть неполными
- Оценка приватных компаний имеет высокую погрешность
- Требуется подключение к интернету
- Некоторые источники могут ограничивать автоматический доступ
